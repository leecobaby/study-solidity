ERC1155 在游戏产业受到了广泛关注。Decentraland，Sandbox 这类红极一时的 Web3 游戏都采用了 ERC1155 标准。

著名网游魔兽世界具有完备的经济系统和道具系统，非常适合 ERC155 施展身手。
今天我就通过实例代码，展示 ERC155 标准如何支持类魔兽世界的金币与道具系统。期望本次的分享能够帮助同学们更好地理解 ERC155 标准。
